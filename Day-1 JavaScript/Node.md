# DataType of JavaScript
 in javascript , like in many programming languages, data types represent the kinds of values you con work with in your code. Here are basic data types

**1. String**
Ans:- A string is a sequence of characters enclosed within single quotes(' '), double quotes(" ") or backtick quotes (` `). String are used to represent a string text

**2. Array**
Ans:- An array is an ordered collection of value.  it can hold values of any data type, including other arrays and objects

**3. Object**
Ans:- Object are a collection of key-value pairs. they are used to complex data structures and represent real-world entities

**4. Number**
Ans:- Number can be integers(Whole Number) or floating-points number(Decimal number).  They are used for arithmetic operations and calculations

**5. Boolean**
Ans:- A Boolean represents one of two values. true, or false. boolean are after used for logical operations and canditinal statements
